Dokumentation

REST-API

nodejs_login SQL database

User Database: (SQL table name = register)

GET - /users för alla användare

GET - /users?id=ID-NUMMER för enskild användare efter ID

Guestbook Database: (SQL table name = comments)

GET - /comments för alla kommentarer

GET - /comments?id=NUMMER för enskild kommentar efter ID

Add new user to Database:

POST - /addUser för lägga till EN användare via Insomnia i JSON-format.

{ "name": "Rudolf", "username": "Rudolf33", "password": "secretpassword", "email": "ilove@emailspam.com" }

POST - /addComment för att lägga till ett gästboksinlägg genom Insomnia i JSON-format.

{ "name": "Pelle", "message": "Mitt meddelande" }

Change thru PUT

PUT - /editUser/ID-NUMMER för att uppdatera en användare med specifikt ID med JSON-format i Insomnia.

{ "name": "Olle", "username": "Olle44", "password": "nopassword", "email": "ilove@emailspam.not" }

PUT - /editComment/ID-NUMMER för att uppdatera ett gästboksinlägg genom Insomnia via ID.

{ "name": "Olle", "message": "Mitt nya fina uppdaterade meddelande" }

DELETE from database!

DELETE - /deleteUser/IDNUMMER - via insomnia eller URL raderar specifik användare med angivet ID

DELETE - /deleteComment/IDNUMMER - via insomnia eller URL för att radera ett specifikt gästboksinlägg ur databasen.
