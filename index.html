Dokumentation

REST-API

nodejs_login SQL database

SQL user table name = register || guestbook table name = comments

Fetch users thru GET:

GET - /users för att hämta alla användare.

GET - /users/ID för att hämta en enskild användare efter ID.

Fetch comments thru GET:

GET - /comments för att hämta alla kommentarer.

GET - /comments/ID för att hämta en enskild kommentar efter ID.

Add new user thru POST:

POST - / Login för att logga in med registrerad användare.

{ "username": "hejsan", "password": "password" }

POST - /addUser för lägga till EN användare via Insomnia i JSON-format.

{ "name": "Rudolf", "username": "Rudolf33", "password": "secretpassword", "email": "ilove@emailspam.com" }

POST - /addComment för att lägga till ett gästboksinlägg genom Insomnia i JSON-format.

{ "name": "Pelle", "message": "Mitt meddelande" }

Change thru PUT:

PUT - /editUser/ID för att uppdatera en användare med specifikt ID med JSON-format i Insomnia.

{ "name": "Olle", "username": "Olle44", "password": "nopassword", "email": "ilove@emailspam.not" }

PUT - /editComment/ID för att uppdatera ett gästboksinlägg genom Insomnia via ID och JSON-format.

{ "name": "Olle", "message": "Mitt nya fina uppdaterade meddelande" }

Remove thru DELETE:

DELETE - /deleteUser/ID - via insomnia eller URL raderar specifik användare med angivet ID

DELETE - /deleteComment/ID - via insomnia eller URL för att radera ett specifikt gästboksinlägg ur databasen.
